StartNotebookInstance

Launches an ML compute instance with the latest version of the libraries and attaches your ML storage volume. After configuring the notebook instance, Amazon SageMaker sets the notebook instance status to InService. A notebook instanceâ€™s status must be InService before you can connect to your Jupyter notebook.

Required: Yes
