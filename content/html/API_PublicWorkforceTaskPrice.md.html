PublicWorkforceTaskPrice

Defines the amount of money paid to an Amazon Mechanical Turk worker for each task performed.

Use one of the following prices for bounding box tasks. Prices are in US dollars. + 0.036 + 0.048 + 0.060 + 0.072 + 0.120 + 0.240 + 0.360 + 0.480 + 0.600 + 0.720 + 0.840 + 0.960 + 1.080 + 1.200

Use one of the following prices for image classification, text classification, and custom tasks. Prices are in US dollars. + 0.012 + 0.024 + 0.036 + 0.048 + 0.060 + 0.072 + 0.120 + 0.240 + 0.360 + 0.480 + 0.600 + 0.720 + 0.840 + 0.960 + 1.080 + 1.200

Use one of the following prices for semantic segmentation tasks. Prices are in US dollars. + 0.840 + 0.960 + 1.080 + 1.200

AmountInUsd Defines the amount of money paid to a worker in United States dollars.
Type: USD object
Required: No
