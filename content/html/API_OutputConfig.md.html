OutputConfig

Contains information about the output location for the compiled model and the device (target) that the model runs on.

S3OutputLocation Identifies the S3 path where you want Amazon SageMaker to store the model artifacts. For example, s3://bucket-name/key-name-prefix.
Type: String
Length Constraints: Maximum length of 1024.
Pattern: ^(https|s3)://([^/]+)/?(.*)$
Required: Yes

TargetDevice Identifies the device that you want to run your model on after it has been compiled. For example: ml_c5.
Type: String
Valid Values:lambda | ml_m4 | ml_m5 | ml_c4 | ml_c5 | ml_p2 | ml_p3 | jetson_tx1 | jetson_tx2 | jetson_nano | rasp3b | deeplens | rk3399 | rk3288 | aisage | sbe_c | qcs605 | qcs603
Required: Yes
